Privacy Policy

Life Tracker is a personal, single-user application operated by and for its sole owner. It is not offered as a service to anyone else.

What data it handles

Daily habit entries the owner logs manually, and — when connected — physiological data retrieved from the owner's own WHOOP account via the WHOOP API (recovery, heart-rate variability, resting heart rate, sleep, strain, and workouts).

Where data lives

Data is stored only in the owner's private database (a personal Supabase project protected by row-level security) and on the owner's own devices. WHOOP data files are excluded from this public code repository.

What it does not do

No data is sold, shared, or transmitted to any third party. There is no advertising, no analytics, and no tracking. No accounts other than the owner's can be created.

Data deletion

The owner can delete all stored data at any time by clearing the database and revoking WHOOP access at app.whoop.com integrations settings.

This application uses the WHOOP API but is not endorsed, certified, or otherwise approved by WHOOP.
